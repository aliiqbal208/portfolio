# Muhammad Ali — Portfolio

Personal portfolio built with **Next.js 16**, **Tailwind CSS v4**, **Three.js**, and **Framer Motion**.

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:3007](http://localhost:3007)

## Tech Stack

- **Framework:** Next.js 16 (App Router, Turbopack)
- **Styling:** Tailwind CSS v4
- **3D:** Three.js, React Three Fiber, Drei
- **Animation:** Framer Motion, GSAP
- **Fonts:** Inter, Bebas Neue, Orbitron, Cormorant Garamond

## Build

```bash
npm run build
npm start
```
